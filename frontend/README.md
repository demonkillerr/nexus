# Nexus Frontend

Next.js 15 + React 19 monitoring dashboard for Nexus.

## Dev

1. Install deps: `npm install`
2. Start dev server: `npm run dev`
3. Open http://localhost:3000

Requires scheduler at http://localhost:8080.

## Prod

Build: `npm run build`

Run: `npm start`

Or use Docker: `docker build -t nexus-frontend .`
